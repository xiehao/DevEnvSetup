# WinDevEnvSetup

本项目包含 Windows 与 macOS 两套独立的自动化配置方案，已按系统拆分到对应目录，避免混淆。

## 目录说明

```text
WinDevEnvSetup/
├── Windows/   # Windows 版本（原脚本与资源）
├── macOS/     # macOS 版本（新脚本与资源）
└── Resources/ # 共享资源（镜像、Dockerfile、配置文件等）
```

## Windows

请进入 `Windows` 目录并按该目录内的 README 操作。

## macOS

请进入 `macOS` 目录并按该目录内的 README 操作。
